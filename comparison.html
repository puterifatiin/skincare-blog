Support Vector Regression (SVR)

skin_svr.py (click here)

import cv2
import numpy as np
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVR
from sklearn.multioutput import MultiOutputRegressor
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score, explained_variance_score
from skimage.feature import hog
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.model_selection import learning_curve

# Load the dataset
df = pd.read_csv('skin_analysis_dataset.csv')


# Function to load and preprocess images from dataset
def load_images(df):
    images = []
    for img_path in df['image_path']:
        img = cv2.imread(img_path, cv2.IMREAD_GRAYSCALE)
        img = cv2.resize(img, (50, 50))  # Resize to 50x50 pixels
        hog_features = hog(img, pixels_per_cell=(8, 8), cells_per_block=(2, 2), visualize=False)
        images.append(hog_features)
    return np.array(images)


# Load images and labels
images = load_images(df)
labels = df[
    ['hydration', 'wrinkles', 'acne', 'pores_size', 'pigmentation', 'redness', 'uniformness', 'eye_area']].values
target_names = ['Hydration', 'Wrinkles', 'Acne', 'Pores Size', 'Pigmentation', 'Redness', 'Uniformness', 'Eye Area']

# Split the dataset into training and testing sets
X_train, X_test, y_train, y_test = train_test_split(images, labels, test_size=0.2, random_state=42)

# Initialize StandardScaler
scaler = StandardScaler()


# Function to evaluate the model
def evaluate_model(y_test, y_pred, model_name):
    mae = mean_absolute_error(y_test, y_pred)
    mse = mean_squared_error(y_test, y_pred)
    rmse = np.sqrt(mse)
    r2 = r2_score(y_test, y_pred)
    explained_variance = explained_variance_score(y_test, y_pred)

    # Handle zero values in y_test for MAPE calculation
    nonzero_mask = y_test != 0
    mape = np.mean(np.abs((y_test[nonzero_mask] - y_pred[nonzero_mask]) / y_test[nonzero_mask])) * 100

    print(f'{model_name} Model Evaluation:')
    print(f'MAE: {mae}')
    print(f'MSE: {mse}')
    print(f'RMSE: {rmse}')
    print(f'R2 Score: {r2}')
    print(f'Explained Variance: {explained_variance}')
    print(f'MAPE: {mape}\n')

    return mae, mse, rmse, r2, explained_variance, mape


# Function to plot predictions vs true values
def plot_predictions(y_test, y_pred, model_name):
    plt.figure(figsize=(10, 6))
    for i, target in enumerate(target_names):
        plt.scatter(y_test[:, i], y_pred[:, i], alpha=0.5, label=target)
    plt.plot([y_test.min(), y_test.max()], [y_test.min(), y_test.max()], 'r--', lw=2)
    plt.xlabel('True Values')
    plt.ylabel('Predictions')
    plt.title(f'{model_name} Predictions vs True Values')
    plt.legend()
    plt.show()


# Function to plot residuals
def plot_residuals(y_test, y_pred, model_name):
    residuals = y_test - y_pred
    plt.figure(figsize=(10, 6))
    for i, target in enumerate(target_names):
        sns.histplot(residuals[:, i], kde=True, label=target, alpha=0.5)
    plt.title(f'{model_name} Residuals Distribution')
    plt.xlabel('Residuals')
    plt.ylabel('Frequency')
    plt.legend()
    plt.show()


# Function to plot learning curves
def plot_learning_curve(estimator, title, X, y):
    train_sizes, train_scores, test_scores = learning_curve(estimator, X, y, cv=5, n_jobs=-1,
                                                            train_sizes=np.linspace(.1, 1.0, 5))
    train_scores_mean = np.mean(train_scores, axis=1)
    train_scores_std = np.std(train_scores, axis=1)
    test_scores_mean = np.mean(test_scores, axis=1)
    test_scores_std = np.std(test_scores, axis=1)

    plt.figure()
    plt.title(title)
    plt.xlabel("Training examples")
    plt.ylabel("Score")
    plt.grid()

    plt.fill_between(train_sizes, train_scores_mean - train_scores_std, train_scores_mean + train_scores_std, alpha=0.1,
                     color="r")
    plt.fill_between(train_sizes, test_scores_mean - test_scores_std, test_scores_mean + test_scores_std, alpha=0.1,
                     color="g")
    plt.plot(train_sizes, train_scores_mean, 'o-', color="r", label="Training score")
    plt.plot(train_sizes, test_scores_mean, 'o-', color="g", label="Cross-validation score")

    plt.legend(loc="best")
    plt.show()


# SVR Model
svr_model = MultiOutputRegressor(SVR(kernel='rbf', C=100, gamma=0.1, epsilon=0.1))

# Standardize features
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled = scaler.transform(X_test)

# Fit the SVR model
svr_model.fit(X_train_scaled, y_train)

# Evaluate the SVR model
y_pred_svr = svr_model.predict(X_test_scaled)
evaluate_model(y_test, y_pred_svr, 'SVR')
plot_predictions(y_test, y_pred_svr, 'SVR')
plot_residuals(y_test, y_pred_svr, 'SVR')
plot_learning_curve(svr_model, 'SVR Learning Curve', X_train_scaled, y_train)

Support Vector Machines (SVM) are commonly employed in machine learning for classification issues, but they may also be used to solve regression problems using Support Vector Regression (SVR). Support Vector Regression (SVR) is a machine learning technique used in regression analysis. The SVR Model in Machine Learning seeks to identify a function that approximates the connection between input variables and a continuous target variable while minimising prediction error.

  • Margin of Tolerance (Epsilon): SVR provides a parameter epsilon (Îµ) that sets a margin of tolerance without penalising mistakes.
  • Hyperparameters: Hyperparameters in SVR include C which is regularisation parameter, kernel type, epsilon, and gamma for kernels such as RBF. Tuning these hyperparameters is critical for optimal performance.

  • Prediction of Skin Properties:
    • - Moisture Levels: SVR be trained to anticipate skin moisture levels using data such as humidity, temperature, and skincare product consumption.
  • Analysis of Skin Metrics:
    • - Wrinkle Formation: SVR can predict the frequency and severity of wrinkle formation based on variables including as age, sun exposure history, and skincare behaviours.
    • - Skin Texture: SVR can assess skin texture using parameters such as pore size, roughness, and elasticity.
  • Enhancing Dermatological Diagnosis:
    • - Once trained, SVR can make predictions about new data inputs. For example, given a collection of skin-related variables, SVR can forecast moisture levels or the possibility of a specific skin condition.

Advantages
  • Accuracy: SVR excels at generalisation and predicts accurately.
  • Ease of use: SVR implementation is easy.
Disadvantages
  • Parameter Tuning: The performance of SVR is strongly dependent on the selection of hyperparameters, which may be complex and time-consuming to optimise. .
  • Scalability: SVR are not suitable for huge datasets.

Random Forest Regression is a flexible machine-learning approach that predicts numerical values. It integrates predictions from many decision trees to prevent overfitting and increase accuracy.

  • Bootstrap Aggregating: RFR is a method for training multiple models using random selections of training data. The forecasts from each model are then pooled, usually by averaging.
  • Stacking: It is predictions from one set of models are used as input characteristics for another model. The second-level model provides the final forecast.

  • Prediction of skin factors:
    • Skin analysis frequently entails predicting factors such as skin moisture content, elasticity, pigmentation levels, and others. RFR can generate models based on training data to reliably predict these values.
  • Feature Selection:
    • RFR is used to find key features from a wide range of variables that help predict skin conditions. This is critical for establishing which parameters such as texture, colour, and moisture levels.

Advantages
  • Scalability: RFR is helpful at managing huge datasets with several properties.
  • Handling: RFR can deal with missing data, outliers, and noisy characteristics.
Disadvantages
  • Computational Complexity: RFR is computationally intensive, particularly with huge datasets.
  • Overfitting: This occurs if the model is too complicated or there are too many decision trees.
